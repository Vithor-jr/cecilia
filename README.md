# Cecilia - Portfólio Profissional

**Cecilia** é um portfólio profissional desenvolvido com **Next.js**, **TypeScript** e **Tailwind CSS**, projetado para apresentar projetos, habilidades e identidade visual de forma moderna, clara e responsiva.

O foco do projeto é unir **performance**, **design refinado** e **manutenibilidade**, utilizando tecnologias amplamente adotadas no mercado e boas práticas de desenvolvimento frontend.

---

## Destaques do Projeto

- Construído com **Next.js**, garantindo excelente performance e SEO
- **TypeScript** para maior segurança, escalabilidade e clareza no código
- Interface estilizada com **Tailwind CSS**, priorizando consistência visual
- Design totalmente responsivo para diferentes dispositivos
- Deploy contínuo e otimizado via **Vercel**

---

## Objetivo

Servir como um **portfólio profissional**, demonstrando:
- Capacidade técnica em frontend moderno
- Organização e qualidade de código
- Atenção a design, usabilidade e experiência do usuário

---

## Como Executar Localmente

### 1. Clone o repositório
```bash
git clone https://github.com/Vithor-jr/cecilia.git
```

### 2. Acesse a pasta do projeto
```bash
cd cecilia
```

### 3. Instale as dependências
```bash
npm install
# ou
yarn install
```

### 4. Inicie o ambiente de desenvolvimento
```bash
npm run dev
# ou
yarn dev
```

### 4. Abra no navegador
```
http://localhost:3000
```
